# Add-on Packs

附屬包提供可選能力。使用時先套用主包，再依需求安裝附屬包。

| Pack | 用途 |
| --- | --- |
| wbs-planning-pack | 產生 WBS、時程、里程碑、驗收項目 |
| executive-presentation-pack | 產生 CIO / 客戶簡報與案例故事 |
| security-scan-pack | 建立白箱、黑箱、弱點掃描流程 |
| enterprise-ui-governance-pack | 建立企業 UI、搜尋、下鑽、管理介面規則 |
| enterprise-db-auth-pack | 建立 MSSQL、AD、Email 預設規則 |
