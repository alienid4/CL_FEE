## 八、系統對應

此簽呈樣本可支援以下系統功能：

- 全文檢索：簽呈號、主旨、承辦人、廠商、採購單號、附件名稱。
- Case 360：顯示簽呈主檔、採購資訊、報價比較、附件、簽核歷程。
- 時間線：簽呈節點可顯示「已核准」，並可點入簽核歷程。
- 舉證鏈：來源截圖 / PDF 頁碼、附件、簽核意見、採購單號。
- 廠商視角：三家廠商報價與利害關係。
- 負責人視角：承辦人與簽核人手上案件。
- 處理優先：若簽呈未核准或卡在某節點，可進入待辦 / 優先矩陣。
