## 2. Excel-first 資料流

```mermaid
flowchart TD
    Excel["既有 Excel<br/>費用項目 / 預算 / 付款 / 續約"] --> Import["匯入批次<br/>import_batches"]
    Import --> Staging["原始列暫存<br/>import_rows + raw_json"]
    Staging --> Mapping["欄位 Mapping<br/>來源欄位 -> 系統欄位"]
    Mapping --> Validate["系統檢核<br/>必填 / 金額 / 日期 / 重複 / 狀態"]
    Validate --> Confirm["人工確認<br/>業助 / 承辦 / 主管"]
    Confirm --> Formal["正式資料<br/>Case / Contract / Budget / Payment"]
    Formal --> Dashboard["Dashboard / A-G 主視圖 / 報表"]
    Formal --> Export["正式 Excel 匯出<br/>保留原範本格式"]
    Export --> FinalUnit["最後單位繳交"]

    PDF["PDF 簽呈"] --> DocStage["文件暫存"]
    DocStage --> TextIndex["文字型 PDF 索引"]
    TextIndex --> SourceRef["來源追溯<br/>檔名 / 頁碼 / 命中段落"]
    SourceRef --> Confirm
```
