# 一次性開發提示詞 4.0 — 從這裡開始

這是「一次性開發提示詞」系列的 **4.0 版 kit**。以 v2.2 為底，繼承其全部 gate 與規則，另加一層「自主迴圈」。

## 三個入口

1. **要用 → 讀這份**：[`一次性開發提示詞_4.0_FINAL.md`](一次性開發提示詞_4.0_FINAL.md)（人類閱讀版，含啟動語，貼上去就跑）
2. **想知道 4.0 加了什麼 → 讀** [`CHANGELOG.md`](CHANGELOG.md) 最上面的 `## 4.0`
3. **底層規則/gate**：其餘檔案（GATE_CATALOG、SECURITY_*、AGENT_AUDIT_RULES…）沿用 v2.2，不放寬。

## 4.0 跟 v2.2 差在哪（一句話）

- v2.2＝**3.5**：一包式開發、gate 齊全，但**每一圈還是人在推**。
- 4.0＝在其上補三塊 → **按一次、走開，它自己跑到底**：
  1. 📋 **待辦自動拉**（做完自己拉下一件）
  2. ⛽ **預算煞車**（回合/時間/失敗到頂就停下出報告）
  3. 🛑 **卡住升級**（同一件失敗 N 次→標 blocked、寫進 decisions.json、停下問人）
- 並綁到真實管線：`.project/snapshot.py`、`.project/checks.py`、`.project/decisions.json`、`.project/review.py`。

## 誠實提醒

目前 `.project/loop.py`（自主驅動器）**尚未建好**。在它建好前，本 kit 讓一個 session **手動達成 4.0 行為**（人按一次觸發、AI 自主跑）。等 `loop.py` 上線，才是真正無人駕駛的 4.0。
