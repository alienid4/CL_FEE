# Enterprise DB Auth

Apply company default DB, auth, and email rules.

## Rules

- Default to Microsoft SQL Server when no DB is specified for formal systems.
- Keep DB access in services or repositories.
- Support AD / LDAP login and role mapping.
- Provide email configuration and test send.
- Never commit real credentials.
