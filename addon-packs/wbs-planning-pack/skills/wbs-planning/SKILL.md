# WBS Planning

Create a practical WBS from project goals.

## Rules

- Use deliverable-oriented tasks, not vague activity names.
- Include start date, end date, owner, dependency, output, and acceptance criteria.
- Separate must-have work from optional polish.
- Call out assumptions and missing decisions.
