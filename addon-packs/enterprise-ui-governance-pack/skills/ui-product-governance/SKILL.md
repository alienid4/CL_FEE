# UI Product Governance

Design enterprise UI for repeated operational use.

## Rules

- Avoid large blank areas.
- Use tabs for subfunctions.
- Make tables sortable.
- Add full-text search.
- Make metric cards drillable.
- If a feature can add records, it must also support edit and delete or disable.
